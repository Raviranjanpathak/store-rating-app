#  Store Rating App

A full-stack web application where users can rate stores (1–5), built as part of a Full Stack Internship Assignment.

##  Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MySQL
- ORM: Sequelize

##  User Roles
- Admin
- Normal User
- Store Owner

##  Features

### Admin
- Dashboard with total users, stores, ratings
- View all users
- Filter users
- View owner ratings

### User
- Signup/Login
- View & search stores
- Submit & update ratings
- Update password

### Store Owner
- View average rating
- See users who rated
- Update password

##  Validations
- Password: 8–16 chars, uppercase + special char
- Name: 20–60 chars
- Email: valid format

##  Setup

### Backend
```bash
cd backend
npm install
npm start

###frontend
